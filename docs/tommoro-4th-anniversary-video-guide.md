## VERSION 1: "여러분 저희 됐어요" — 회사 4년 성과 총정리

> 길이: 40~50초
> 톤: 에너지 폭발, 밈 그대로
> 대상: 전체 회사

### Audio

- **나레이션**: 흥분된 목소리로 "쌰갈 여러분 저희 됐어요 됐다고요!!"
  - 직접 녹음 권장 (밈 특유의 흥분 톤이 핵심)
  - 또는 TTS + 피치 올리기
- **BGM**: 에너지 넘치는 트랩 비트 또는 에픽 드롭 (비트 드롭에 컷 전환 맞추기)
  - 추천: 유행 릴스 오디오 활용 또는 CapCut 내장 BGM

### Production Notes

- **AI 생성 활용**: 정적 이미지를 Runway/Kling으로 미세 모션 추가 (줌인, 패럴렉스 등)
- **공감 연출 핵심**:
  - CES 컷 (7~8): 비행기 티켓 → 부스 세팅 → 데모 성공 순서로 "출장 서사" 느낌
  - Habilis-β 컷 (11): 시계 빨리감기 + 로봇 반복 작업 → 숫자 카운터 올라가는 그래픽
  - WUJI Hand 컷 (12): 사람 손과 로봇 손 나란히 놓는 구도, AI로 손가락 움직임 생성
- **컷 전환**: 비트에 맞춰 하드컷 또는 스와이프 전환 (릴스 느낌)
- **자막**: 굵은 흰색 + 검정 아웃라인, 화면 중앙 하단
- **핵심**: 나레이션의 "됐어요" 톤이 영상의 생명 — 최대한 흥분되게
