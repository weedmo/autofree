# weed-harness for Codex

This package exposes copied `skills/` and `agents/` content from the `autofree` repository for Codex plugin installation.

Use the packaged skills as workflow guidance for coding, research, QA, and release work. When a skill mentions Claude-specific hooks, slash commands, or `~/.claude` paths, treat that as Claude-oriented guidance rather than a guaranteed Codex runtime feature.

This Codex package does not ship the repository's Claude hook automation.
